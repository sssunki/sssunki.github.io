在 Android 平台上利用 FFmpeg 给 Gif 图片加字幕

发表于 2020-09-14 更新于 2020-09-16

复刻了一下表情锅的技术方案，利用 ffmpeg 给 gif 图片添加字幕。

前置知识：JNI、交叉编译，FFmpeg以及流媒体相关基础知识。（本文会有简单介绍）

概念介绍

  1. JNI：
  2. 交叉编译
  3. FFmpeg
  4. ass 文件
  5. 流媒体相关

实现思路

在 PC 上使用 FFmpeg 添加字幕的基本方式是这样的

ffmpeg 的各种功能都是在对音视频流进行处理，所以不能直接处理 gif，需要把 gif 转换称 mp4 媒体文件
ffmpeg -f gif -i out.gif ss.mp4

使用 ass 文件给 mp4 添加字幕
ffmpeg -i ffmpegtest.MP4 -vf ass=subtitles.ass out.mp4

最后再把 mp4 转换成 gif
ffmpeg -i out.mp4 out.gif
